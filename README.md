## 环境与 Demo 说明

- 本仓库所有 Demo 已在 **Qt 6.4.3** 环境下测试通过

## 开发建议

- 涉及 QML + C++ 混合开发时，**强烈建议使用 Visual Studio 编写 C++ 代码**  
- 配合 Qt VS Tools 可获得更好的调试、补全与工程管理体验
